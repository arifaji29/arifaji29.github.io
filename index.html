About Me

Hallo saya Arif, Saya adalah seorang Desainer Grafis dan Website Programmer. Saya lahir di Pekalongan pada tanggal 29 Maret 1997. Saat ini saya tinggal di Pekalongan Jawa Tengah. Saya Tidak memiliki background pendidikan IT maupun Multimedia. Namun saya memiliki semangat untuk belajar Desain Grafis dan Programming melalui Kursus, Pelatihan ataupun belajar secara otodidak melalui berbagai sumber belajar.

Saat ini saya bekerja sebagai Desainer Grafis di Salah satu brand clothing di Pekalongan. Selain Desain Grafis saya juga tertarik pada bidang Web Programming dan saat ini saya masih terus belajar teknologi-teknologi front-end maupun back-end web developing secara online. Teknologi website yang saya kuasai sekarang anatara lain : html, css, bootstrap, Javascript, React
