About

Hallo saya Arif, Saya adalah seorang Desainer Grafis dan Website Developer. Saya lahir di Pekalongan pada tanggal 29 Maret 1997. Saat ini saya tinggal di Pekalongan Jawa Tengah. Saya menamatkan studi S1 pada tahun 2019 di Universitas Negeri Semarang Jurusan Pendidikan Non Formal. Saya menguasai beberapa software desain diantaranya : Corel Draw, Adobe Photoshop, Canva.

Saat ini saya bekerja sebagai Desainer Grafis di Salah satu brand clothing di Pekalongan. Selain Desain Grafis saya juga tertarik pada bidang Web Programming dan saat ini tengah belajar di salah satu Bootcamp fullstack web developer secara online. Teknologi website yang saya kuasai sekarang anatara lain : html, css, bootstrap, Javascript, dan php.
